# 真如苑資料網站（非官方）

全新莊嚴禪意深色風格的資料彙整網站，含**前台展示**與**後台維護**，資料儲存採用 **Google Apps Script (GAS) + Google 試算表**。

參考重建自：<https://sites.google.com/view/shinnyoc>

---

## 一、檔案結構

```
41355/
├─ index.html              前台首頁（單頁式，含五大區塊）
├─ admin.html              後台維護介面（登入 + 增刪改）
├─ assets/
│  ├─ css/style.css        主站樣式（莊嚴禪意深色主題）
│  ├─ css/admin.css        後台樣式
│  └─ js/
│     ├─ config.js         ★ 設定檔：填入 GAS_URL
│     ├─ api.js            API 客戶端（含展示模式範例資料）
│     ├─ app.js            前台渲染
│     └─ admin.js          後台 CRUD 邏輯
├─ gas/
│  └─ Code.gs              ★ GAS 後端（複製到 script.google.com）
└─ README.md
```

> 目前未設定 GAS 時，網站會自動進入**展示模式**，直接以 `assets/js/seed-data.js`（已從原站移轉的真實資料）呈現（後台僅能預覽，無法儲存）。
>
> **資料移轉**：已將原站全部真實資料整理進專案 —— Podcast 16 集、親苑時報 114 期、台灣行事曆 8 期、瑞聲法語 48 號。執行 GAS `setup()` 時會一次寫入試算表；亦可用 `sheets/*.csv` 手動匯入。

## 二、後端 GAS 部署（約 5 分鐘）

1. 前往 <https://script.google.com> →「新增專案」。
2. 把 `gas/Code.gs` 全部內容貼進去，覆蓋預設的 `Code.gs`，存檔。
3. 在上方函式選單選擇 **`setup`** → 按「執行」。
   - 第一次會要求授權，請允許（建立試算表、讀寫權限）。
   - 執行成功後會自動建立一份名為「真如苑資料網站 — 資料庫」的試算表，並寫入範例資料。
4. 右上角 **部署 → 新增部署作業**：
   - 類型：**網頁應用程式**
   - 執行身分：**我**
   - 具有存取權的使用者：**任何人**
   - 按「部署」，複製產生的 **網頁應用程式 URL**（以 `/exec` 結尾）。
5. 打開 `assets/js/config.js`，把該網址貼到 `GAS_URL`：
   ```js
   GAS_URL: 'https://script.google.com/macros/s/AKfycb..../exec',
   ```
6. 重新整理前台與後台即可連線。

### 預設管理密碼
- 初始密碼：**`shinnyo2026`**
- 登入後台後請立刻點右上角「**修改密碼**」更換。
- 密碼以 SHA-256 加鹽雜湊存於 Script Properties，不存明碼。

## 三、前台部署（nginx）

本專案已放在 nginx 的 `html/41355/` 下，為純靜態檔案，直接以網址存取 `index.html` 即可，無需額外建置。

---

## 四、後台維護功能

進入 `admin.html` → 輸入密碼登入後，左側可切換五大資料類型，皆支援**新增 / 編輯 / 刪除**：

| 類型 | 主要欄位 |
|------|----------|
| 最新消息 | 標題、日期、內容、連結、置頂、排序 |
| Podcast | 集數、標題、來賓、日期、簡介、連結、封面、排序 |
| 行事曆 | 日期、活動名稱、地點、分類、說明、排序 |
| 親苑時報 | 期別、標題、日期、PDF連結、封面、排序 |
| 瑞聲法語 | 標題、分類、日期、內容、連結、排序 |

- **排序**：數字越小越前；相同則依日期由新到舊。
- 資料即時寫入 Google 試算表，可直接在試算表查看／批次編輯，前台會同步顯示。

---

## 五、安全性說明

- 讀取（前台顯示）為公開 API。
- 新增／修改／刪除／改密碼皆需先登入取得 token（6 小時有效，存於 CacheService）。
- 若需更高安全性，可改用 Google 帳號授權（`Session.getActiveUser()`），請告知以調整。

---

## 六、自訂

- **站名／副標／官方連結／頁尾**：改 `assets/js/config.js`。
- **配色**：改 `assets/css/style.css` 最上方 `:root` 變數（金色 `--gold-*`、底色 `--ink-*`）。
- **新增欄位**：在 `gas/Code.gs` 的 `SCHEMA` 加欄位，並在 `assets/js/admin.js` 的 `COLLECTIONS` 對應 `fields` 加上即可。
